À propos de moi

Bonjour, je m'appelle Christian Gram. Actuellement étudiant à Ensign College, je me spécialise dans le domaine de l'informatique (IT) et je me passionne particulièrement pour le développement web.

Depuis que j'ai découvert le monde de la programmation, je me consacre à apprendre et à créer des sites web dynamiques et interactifs. J'aime combiner la logique du code avec le design pour construire des expériences utilisateur intuitives et esthétiques.

Mes compétences actuelles incluent :

    HTML, CSS et JavaScript pour la création de sites web modernes Node.js et Express pour le développement de sites dynamiques côté serveur Gestion de versions avec Git et GitHub pour organiser et collaborer sur mes projets

Je suis toujours à la recherche de nouveaux défis pour améliorer mes compétences et élargir mes connaissances en développement web. Mon objectif est de devenir un développeur web compétent capable de concevoir des sites professionnels et innovants.

Si vous voulez suivre mon parcours ou voir mes projets, n'hésitez pas à explorer les autres sections de ce site.
